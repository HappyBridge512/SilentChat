# Silent Duo Chat

Анонімний веб-чат без реєстрації для **двох** користувачів.

## Можливості
- Створення кімнати з одноразовим invite-посиланням
- Лише 2 учасники в кімнаті
- Текстові повідомлення в реальному часі
- Надсилання файлів і фото
- Після виходу будь-кого чат завершується
- Історія чату та завантажені файли видаляються після завершення кімнати

## Запуск
```bash
npm install
npm start
```

Відкрий `http://localhost:3000`.

Якщо порт `3000` зайнятий:

```bash
npm run start:clean
```

## Доступ з іншої мережі (інтернет)
Запуск через публічний тунель:

```bash
npm run start:public
```

Після старту в консолі з'явиться рядок:
`Public tunnel URL: https://...`

Використовується Cloudflare Tunnel (`trycloudflare.com`) без сторінки з tunnel password.
Саме цей URL буде використовуватись у запрошеннях кімнат (`inviteUrlPublic`).

Якщо порт зайнятий:

```bash
npm run start:public:clean
```

## Приватність
- Додані anti-cache headers (`Cache-Control: no-store`, `Pragma: no-cache`, `Expires: 0`)
- Прострочені кімнати автоматично завершуються (TTL за замовчуванням 60 хв)
